Actividad previa. Instalación de Packet Tracer
.
.

      Objetivo:  
    • Preparar el entorno de trabajo que estaremos utilizando a lo largo de este curso.

   Modalidad:  
    • Individual.
 
     Instrucciones:
    • Utiliza tu cuenta y password de netacad.com y realiza la instalación del programa Packet Tracer en tu computadora personal.  La versión de este software está disponible para plataforma Windows y Mac. Para verificar que el software quedó apropiadamente instalado, ejecuta la aplicación de Packet Tracer utilizando los datos de tu cuenta de netacad.  La verificación de tus credenciales sólo se realizará una vez, por lo que es importante completar el proceso de configuración de lo contrario las opciones para salvar tus diseños y configuraciones de Packet Tracer no quedarán habilitadas.
    • Realiza una impresión de pantalla en la que se visualice la ejecución del software instalado.
    • Utiliza un editor de texto para crear un documento que contenga una portada con los datos del curso, tus datos personales y la impresión de pantalla que demuestra la instalación correcta del software.
    • Salva la última versión de tu documento en formato PDF. Esta es la versión final que subirás como evidencia de la realización de esta actividad.

       Especificaciones de entrega
 
    • Formato de entrega: Un archivo pdf
    • Nombre de los entregables: PT_matrícula
    • Medio de entrega: Se entrega en la sección de Actividad previa: Instalación de Packet Tracer.
